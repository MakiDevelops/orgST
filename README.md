# install and run
-- -- -- -- -- --
Install the latest jar, then:
Open either Microsoft PowerShell (for windows) or the terminal (for Macos and Linux)
Enter the directory the jar is in using `cd {exampledir}`
Then use: `java -jar {jarnamehere}.jar`
