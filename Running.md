# How To Run
>[!IMPORTANT]
> Java 21 or later is required!
  

> __For Windows__  
>Download [orgST.jar](https://github.com/MakiDevelops/orgST/raw/refs/heads/main/target/orgST.jar)  
>Open Microsoft PowerShell and Do
```sh
cd %USERPROFILE%\Downloads
java -jar orgST.jar
```
  
> __For macOS & Linux__  
> Download [orgST.jar](https://github.com/MakiDevelops/orgST/raw/refs/heads/main/target/orgST.jar)  
> Open Terminal and Do
```sh
cd ~/Downloads
java -jar orgST.jar
```
